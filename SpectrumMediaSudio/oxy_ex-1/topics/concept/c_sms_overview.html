Overview

Spectrum Media Studio is a software application specifically designed to control the Harmonic Spectrum X Media Server. Its panel-based design brings together ingest, playout, graphics, switching, audio, and macro control in a single user interface.

Using the Spectrum Media Studio software application, users can manually control channel-in-a-box capabilities for a wide range of studio and master control-oriented workflows. It allows broadcasters to ingest content from live and tape sources; control single and multi-channel playout for on air and QC operations; import and creation of playlists, scheduled recordings, multi-layer graphics, audio profiles and subtitles for live and file-based content. The software features a modern, fully customizable, and touchscreen-compatible user interface that supports Spectrum X media server features, now and in the future.

Users can effortlessly swap between different desktops to accommodate multiple workflows, and facilities can leverage new features as their business and workflow requirements evolve.
