Snapshots

Spectrum Media Studio allows the creation of unlimited snapshots.

Note: From release 1.4.x.x. the following limitations apply to snapshots:
  1. Minimum resolution is 1920 x 1080
  2. When using multiple monitors, Spectrum Media Studio is only supported on the “primary” monitor when creating snapshots. Creating snapshots on other monitors will give unpredictable results.

Every snapshot can be associated with up to 4 desktops.

The snapshot is a working environment.

  • Configuration:

    Click on the button "save" to display the Snapshot Manager panel.

    Click on the button "plus" to add a new snapshot.

    Click on the button "bin" to erase the selected snapshot.
    • Desktop 1 to 4: Define a name for each desktop associated to the snapshot.
    • Name: Define the name of the channel assigned to the snapshot.
    • Icon: Select the icon of the channel. The icon is a png file that has to be placed in the folder: C:\Services Spectrum Media Studio\SMS_V1.7\Config\Snaps\Icons, for example.
    • Colors: Set the color of the background. It can be used to distinguish at a glance the snapshot currently being displayed.
    • Image: Load a picture that can be used as the desktop background, instead of the color option above.

Desktops

To switch between desktops, use the buttons at the top bar.

Note: Configuration of a desktop depends on the Admin rights

Spectrum Media Studio offers the possibility to lock / unlock a desktop configuration.

Lock the desktop to avoid any accidental control changes.

Click on the locker icon at the bottom right of the desktop to lock or unlock it.

File Intelligent Transfer

The File Intelligent Transfer service is a service provided by Spectrum Media Studio which can be configured to connect to various storage silos, either Spectrum, MediaGrid or a NAS. The FiT service must be configured to allow clips to be played by the players and playlist panels in Spectrum Media Studio. It is also possible to transfer clips between the configured storage silo's when using an FTP connection. The Spectrum can also "Preview" clips from a MediaGrid or NAS it is licensed and configured to do so. Please see the Spectrum and System Manager guides for configuration, licensing and limitations when using preview.

    • Green state: connected to the service.
    • Red state: the Spectrum Media Studio is not connected to FIT.
Note: If the Spectrum Media Studio is not connected, refer to the section Installing FIT ServiceXXXXXXXXXXXXXXXXXXXXXXXXX

Heartbeats

The Heartbeat status is created to monitor the health of each server connection.

  • Green state: all servers are available.
  • Red state: at least one server is not responding.
