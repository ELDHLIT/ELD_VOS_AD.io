Ordering Guide

Commercial Reference Description
MEDIA-STUDIO-BASE Base license. Required. Base system includes: - Windows Application, one player license can be used for playback or recording. Recorder supports crash, scheduled or import of record list. VTR record module. Single player playlist module. Single sub clip creation. Import of Polaris Play Ingest playlist. Customer supplies PC.
MEDIA-STUDIO-CH Optional license. License to add one extra channel Player or Recorder to Spectrum Media Studio Base. Requires MEDIA-STUDIO-BASE.
MEDIA-STUDIO-MULTIPL Optional license. License to add multiplayer playlist option to a desktop. Up to 4 players per multiplayer desktop supported. Individual player licenses are required to be purchased for each player required. Requires MEDIA-STUDIO-BASE.
MEDIA-STUDIO-PL-IMPORT Optional license. License to add ability to import currently available playlist formats. See release notes for currently supported formats. Customers can also purchase this license to request a translator to be created and added to the supported list if your playlist is not supported at this time - Please contact PM to confirm availability.
MEDIA-STUDIO-GFX-CH Optional license. License to add one connection for MCS Switcher, GFX Keyer and Audio profile control connection. The GFX Keyer supports up to 8 layers and each layer can be pre-configured to control a specific GFX template. Requires MEDIA-STUDIO-BASE.
MEDIA-STUDIO-GFX-AD Optional license. License adds the ability to create Macros and more complex and dynamic control of the GFX keyers. Requires MEDIA-STUDIO-BASE and MEDIA-STUDIO-GFX-CH.
MEDIA-STUDIO-STLFILE Optional license. License to add the ability to manage STL Subtitle files, preview STL files and Preview STL files, including ability to view in sync with playing video. (Video requires SDI port on Spectrum). Requires MEDIA-STUDIO-BASE.
MEDIA-STUDIO-STLLIVE Optional license. License to add control of incoming LIVE subtitle streams. Adds the ability to connect/disconnect LIVE incoming streams to required live input on the Spectrum. Note: this function may require custom development and testing. Please enquire before selling this option. Requires MEDIA-STUDIO-BASE.
MEDIA-STUDIO-KANTAR Optional license. License to add the ability to control the Spectrum internal Kantar water marking feature. Requires MEDIA-STUDIO-BASE and Kantar license for Spectrum. Does not include Kantar watermarking service.
MEDIA-STUDIO-PL-IMPORT Optional license. License to add the ability to import currently available playlist formats.
MEDIA-STUDIO-ROUTE-SVC Optional license. License for single router service. A single router service can manage 1 router. See documentation for supported protocols.
MEDIA-STUDIO-ROUTE-CLIENT Optional license. License for one client connection to the MEDIA-STUDIO-ROUTE-SVC.
