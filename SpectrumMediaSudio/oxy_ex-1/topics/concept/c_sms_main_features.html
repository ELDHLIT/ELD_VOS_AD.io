Main Features

Spectrum Media Studio is a standalone application that runs on a Windows 10 operating system.

Spectrum Media Studio offers the following main features.

Features Description
Simple Player control for QC operations Spectrum Media Studio users can configure a single player allowing them to access clips on the local Spectrum, to view in real time and/or shuttle. Users can add Mark In and Mark Out points and/or create sub clips.
Single channel playlist control, with follow, manual and timed operation Spectrum Media Studio users can create multi-clip playlists or import previously created playlists and control either each clip manually, clips back to back or have the playlist start at a fixed predefined time.
Multi-Player Panel Spectrum Media Studio users can attach up to four players to a single panel. Each player has access to the Spectrum media or playlist. Each player is controlled independently and can be used manually or back to back playlist mode.
Crash and simple scheduled record control Spectrum Media Studio recorder panels allow users to perform instant records, or setup predefined scheduled recordings.
GFX keyer control Spectrum Media Studio users can configure keyer bars to control the GFX layers on each Spectrum Channel.
Ability to define Macros for more complex operations Macros allow users to define more complex and dynamic buttons to control GFX, along with control of the MCS switcher, player, and Audio profiles. This allows multiple operations to be carried out at the touch of a button.
MCS control Create panels to control the MCS switcher within each Spectrum, either standard or enhanced channels.
Audio profile control Operators can define panels that allow selection of the Spectrum audio profiles for all the sources on the MCS switcher.
VTR Ingest Control Users can control and Ingest from external VTR devices to the Spectrum server.
Note: Please contact Harmonic for supported framerates.
Snapshots/Desktops Users can define and customize snapshots. Each snapshot can contain up to four desktops, with each desktop containing the required operational panel. Users can switch between desktops and snapshots as the required workflow changes.
Live subtitle operation With Spectrum Media Studio users can preview incoming live subtitle streams and attach/detach them to the required incoming live feeds on the Spectrum.
Note: Live subtitling workflows require prior workflow consultation and testing.
Preview/ edit subtitle files Users can preview and edit .STL subtitle files that exist on the Spectrum file system.
Router Control Spectrum Media Studio can remotely control a router via the Router Service. This is currently limited to the recorder panel. See documentation for supported routers.

Spectrum Media Studio native integration allows Harmonic to offer a compelling suite of functions with its top of the range Video Server, giving customers the option to purchase a complete solution for many workflows.
